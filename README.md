# Shiny

Application Shiny pour analyse descriptive et modélisation statistique développée.

## Contenu du projet
- `global.R` : packages et fonctions/modèles
- `ui.R` : interface utilisateur
- `server.R` : logique serveur
- `www/` : fichiers statiques (logo, CSS)

## Objectif
Permet de charger un jeu de données, visualiser, décrire et modéliser les variables.
